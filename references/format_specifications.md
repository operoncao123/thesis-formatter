## 页面设置

### 纸张大小
- **纸张**: A4 (210mm × 297mm)
- **DXA单位**: 12240 × 15840

### 页边距
- **上边距**: 3.0 cm (1701 DXA)
- **下边距**: 2.5 cm (1417 DXA)
- **左边距**: 2.6 cm (1474 DXA)
- **右边距**: 2.6 cm (1474 DXA)
- **装订线**: 0 cm，位置在左侧
- **左右对称页边距**: 是

### 页眉页脚距离
- **页眉距边界**: 2.0 cm (1134 DXA)
- **页脚距边界**: 1.75 cm (1.75 cm = 992 DXA)

## 字体字号规范

### 中文字体
- **正文**: 宋体
- **标题**: 黑体

### 英文字体
- **正文英文、数字、符号**: Times New Roman

## 标题格式

### 章标题（如"第一章 文献综述"）

```
字体: 黑体
字号: 三号 (16pt)
对齐: 居中
行距: 单倍行距
段前: 24磅
段后: 18磅
格式: 章序号用中文数字，与标题间空一个汉字符
示例: 第一章 文献综述
```

### 一级节标题（如"1.1 DNA损伤修复"）

```
字体: 黑体
字号: 四号 (14pt)
对齐: 左对齐
行距: 固定值20磅
段前: 24磅
段后: 6磅
格式: 标号与标题间空一个英文字符
示例: 1.1 DNA损伤修复
```

### 二级节标题（如"1.1.1 DNA双链断裂的修复"）

```
字体: 黑体
字号: 13pt
对齐: 左对齐
行距: 固定值20磅
段前: 12磅
段后: 6磅
格式: 标号与标题间空一个英文字符
示例: 1.1.1 DNA双链断裂的修复
```

### 三级节标题（如"1.1.1.1"）

```
字体: 黑体
字号: 小四号 (12pt)
对齐: 左对齐
行距: 固定值20磅
段前: 12磅
段后: 6磅
注意: 一般不建议使用三级及以上节标题
```

### 表格格式

**CRITICAL: 北大博士论文使用三线表格式**

三线表只有三条线：
1. 表格顶线（粗线，10pt）
2. 表头下方线（细线，6pt）
3. 表格底线（粗线，10pt）

**不要使用**：
- 内部横线
- 任何竖线
- 单元格边框

```
表序与表名位置: 表的上方
字体: 宋体 (中文)、Times New Roman (英文/数字/符号)
字号: 11pt
表序: 粗体
段前: 12磅
段后: 6磅
行距: 单倍行距
对齐: 居中
表序与表名间隔: 一个汉字符

格式示例:
表2.1 分子与生化试剂
```

#### 表格内文字
```
字体: 宋体 (中文)、Times New Roman (英文/数字/符号)
字号: 11pt
行距: 单倍行距
段前: 3磅
段后: 3磅
对齐: 表头居中，内容列左对齐
```

#### 续表格式

当表格跨页时：

## 图表格式

### 图片格式

```
图序与图注位置: 图的下方
字体: 宋体 (中文)、Times New Roman (英文/数字/符号)
字号: 11pt
标题: 粗体
段前: 6磅
段后: 12磅
行距: 单倍行距
对齐: 两端对齐
图序与标题间隔: 一个英文字符

格式示例:
图3.1 七重突变体tcp2/3/4/5/10/13/17形成具有绿色花瓣的花。A. 野生型成熟花瓣。B. tcp2/3/4/5/10/13/17七重突变体成熟花瓣。为方便观察花瓣颜色分布，图中所展示的花是去除了花中的萼片和雄蕊的照片。比例尺为1 mm。
```

#### 图中文字要求
- **字号**: 6-9pt
- **中文字体**: 黑体
- **英文字体**: Arial
- **分图标记**: A、B、C...（10pt英文字体，放在分图左上角）

#### 图片质量要求
- 分辨率足够高，400%显示时清晰不模糊
- 图注应详细清楚，具有自明性
- 如果图片来自文献，需标明出处

### 表格格式

```
表序与表名位置: 表的上方
字体: 宋体 (中文)、Times New Roman (英文/数字/符号)
字号: 11pt
表序: 粗体
段前: 12磅
段后: 6磅
行距: 单倍行距
对齐: 居中
表序与表名间隔: 一个汉字符

格式示例:
表3.5 室外细菌气溶胶香农-维纳指数（H）和均匀性指数（E）
```

#### 表格内文字
```
字体: 宋体 (中文)、Times New Roman (英文/数字/符号)
字号: 11pt
行距: 单倍行距
段前: 3磅
```

## 参考文献格式

### 基本要求（顺序编码制）

1. **著录原则**
   - 论文中引用过的文献必须著录
   - 未引用的文献不得出现
   - 不重复罗列
   - 按在论文中出现的顺序编号

2. **正文中引用格式**
   - 序号放在方括号内：[1]
   - 以上标方式标注在句尾
   - 示例：...已有研究表明[1,2]。

3. **参考文献列表格式**
   ```
   字体: 宋体 (中文)、Times New Roman (英文)
   字号: 五号 (10.5pt)
   行距: 固定值16磅
   段前: 3磅
   段后: 0磅
   格式: 悬挂格式，序号悬挂在外
   标点: 使用英文标点符号
   ```

### Nature格式详细说明

#### 英文期刊论文格式

```
序号. 作者1, 作者2, 作者3 & 作者4. 文章标题. 期刊名 卷号, 起始页–终止页 (年份).

示例:
1. Banani, S. F., Lee, H. O., Hyman, A. A. & Rosen, M. K. Biomolecular condensates: organizers of cellular biochemistry. Nat. Rev. Mol. Cell Biol. 18, 285–298 (2017).
```

**格式要点:**
- 作者姓名用逗号分隔
- 最后两个作者用"&"连接
- 期刊名用斜体（在XML中用`<w:i/>`）
- 卷号用粗体（在XML中用`<w:b/>`）
- 页码用短横线（–）连接
- 年份放在括号内

#### 中文期刊论文格式

```
序号. 作者1, 作者2, 作者3, 作者4. 文章标题. 期刊名, 卷号, 起始页–终止页 (年份).

示例:
段凤魁, 贺克斌, 刘咸德, 董树屏, 杨复沫. 含碳气溶胶研究进展：有机碳和元素碳. 环境工程学报, 1, 1–8 (2007).
```

**格式要点:**
- 中文作者超过5个用"第一作者名等"
- 格式与英文文献类似
- 中英文文献混排，按编号从小到大排列

## 页眉页脚

### 页眉设置

**页眉范围**: 从"摘要"开始到论文结束

**奇数页页眉**:
- 内容: 当前章节名称（如"第一章 文献综述"）
- 字体: 宋体
- 字号: 五号 (10.5pt)
- 对齐: 居中

**偶数页页眉**:
- 内容: "YOUR_UNIVERSITY博士学位论文"
- 字体: 宋体
- 字号: 五号 (10.5pt)
- 对齐: 居中

### 页码设置

**摘要和目录**:
- 格式: 大写罗马数字（I, II, III, IV...）
- 位置: 页脚居中

**正文到致谢**:
- 格式: 阿拉伯数字（1, 2, 3, 4...）
- 位置: 页脚居中

**不编页码的部分**:
- 封面
- 版权声明
- 原创性声明和使用授权说明

## 特殊要求

### 化学式和符号

- **上下标**: CO₂（使用`<w:vertAlign w:val="subscript"/>`）
- **希腊字母**: α、β、γ、μ等用Times New Roman
- **单位符号**: μM、pH、°C等格式正确
- **间距**: %和°C与数字间不留空格，其他单位留一个空格

### 标点符号

- 中文行文使用中文标点
- 避免出现英文标点（如" "）
- 标题序号与标题间空一个空格
- 数字与单位间空一个空格（%和°C除外）

### 目录格式

```
章标题: 黑体小四号 (12pt)，固定行距20磅，段前6磅，段后0磅
其他内容: 宋体小四号 (12pt)，固定行距20磅，段前、段后0磅
英文/数字/符号: Times New Roman
对齐: 章标题左对齐，一级节标题缩进1个汉字符，二级节标题缩进2个汉字符
点符号: 标题和页码之间用点符号连接
```

### 论文组成部分顺序

1. 封面
2. 版权声明
3. 中文摘要
4. 英文摘要
5. 目录
6. 缩略词或符号表（如有）
7. 正文
8. 参考文献
9. 附录（如有）
10. 个人简历
11. 致谢
12. 学位论文原创性声明和授权使用说明

### 论文长度要求

- **博士论文**: 到参考文献前页数不少于100页
- **封面**: 采用北大研究生院指定的青绿色封面
