# Skill更新说明 - 2026-03-13

## 更新来源

根据标准博士论文模板 PDF 学习并整合了新的格式要求。

## 主要更新内容

### 1. 表格格式规范（重要更新）

**发现问题**：之前的skill没有正确处理表格格式，使用了普通表格而非北大要求的三线表。

**新增内容**：
- **三线表格式**：只有顶线（粗10pt）、表头下方线（细6pt）、底线（粗10pt）
- **无竖线**：所有竖线都不显示
- **无内部横线**：表格内容行之间无横线
- **续表格式**：跨页表格需要标注"（续表）"并重复表头
- **表格内容格式**：宋体11pt，单倍行距，段前3磅，段后3磅

**新增工具**：
- `scripts/convert_to_three_line_table.py` - 自动将表格转换为三线表格式

### 2. 标题识别规则（重要更新）

**发现问题**：在测试文件中，错误地将某些实验方法名称（如带有具体操作名称的小节标题）识别为章标题，但它们实际上是某章下的小节内容。

**新增内容**：
- **章标题识别规则**：必须有"第X章"字样，如"第一章 文献综述"、"第二章 实验材料和方法"
- **一级节标题**：使用数字编号如"1.1"、"2.1"
- **二级节标题**：使用三位数字编号如"1.1.1"、"2.1.1"
- **常见错误示例**：材料、方法、实验步骤等通常不是独立的章，而是"实验材料和方法"章的内容

**更新位置**：
- `SKILL.md` - 添加了"标题层级识别规则"章节
- 包含详细的识别方法和常见错误示例

### 3. 表格格式详细说明

**更新文件**：
- `SKILL.md` - 更新了3.4节"图表格式"
- `references/format_specifications.md` - 添加了详细的三线表XML格式说明
- `README.md` - 更新了表格格式说明

**新增内容**：
- 三线表的XML边框设置代码
- 表头行、内容行、最后一行的不同边框设置
- 续表的完整格式要求
- 表格示例（文本形式展示三线表效果）

### 4. 从PDF模板学到的其他细节

**表格标注格式**：
- 表2.1 分子与生化试剂
- 表2.1 分子与生化试剂（续表）

**表格内容对齐**：
- 表头：居中
- 内容列：左对齐

**表格单元格格式**：
- 字体：宋体（中文）、Times New Roman（英文/数字）
- 字号：11pt (w:sz="22")
- 行距：单倍行距
- 段前：3磅
- 段后：3磅

## 使用说明

```bash

python scripts/office/unpack.py input.docx unpacked/

# 2. 转换表格为三线表
python scripts/convert_to_three_line_table.py unpacked/word/document.xml

# 3. 应用其他格式（标题、字体等）
# ... 使用Edit工具修改XML

# 4. 重新打包
python scripts/office/pack.py unpacked/ output.docx --original input.docx
```

### 验证格式

```bash

```
